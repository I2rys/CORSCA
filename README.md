## Installation
Github:
```
git clone https://github.com/I2rys/CORSCA
```

NPM Packages:
```
npm i axios
```

## Usage
```
node index.js <input> <output>
```
+ input - The file path that contains websites links.
+ output - The output of the websites links that is vulnerable to CORS.
